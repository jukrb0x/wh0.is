本文将介绍一个带有图形界面的虚幻引擎 (Unreal Engine) 编辑器插件的开发过程，包含以下内容：

-   编辑器拓展基本知识
-   如何创建一个插件
-   Slate UI 和 Slate Widget 的基本使用
-   事件委托 (Delegate)
-   UE 反射系统 (Reflection) 以及简单应用

受限于笔者技术水平，本文不是一个详尽的 UE 编辑器教程，仅为一些开发过程的经验记录，请同时参阅对应的引擎 API 和官方文档。

文中使用的引擎版本为 `5.1.1` 源码编译版，可能些许地方的逻辑和 Epic Games Launcher 安装版会有些不同。使用 `4.x` 或 `5.x` 的引擎原理和逻辑上大同小异，获取源代码的方式可以虚幻官方文档中找到。

Module, 以 LyraStarterGame 为例：
